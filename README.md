# lnbitsrustclient

An API Client for [LNbits](https://lnbits.com/) in Rust.

## Key features

- [x] Create invoices
- [x] Webhook for incoming invoices
- [x] Decode invoices
- [x] Pay invoices
- [x] Get wallet details
- [x] Boltz
